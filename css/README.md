This folder contains modular CSS for the portfolio:

- general.css: reset, shared components, buttons, common section styles
- body.css: base body styles
- header.css: navigation bar and logo styles
- hero.css: hero section, h1, subtitles, CTA
- skills.css: skills grid and cards
- projects.css: project grid and cards
- education.css: education box and info grid
- hobbies.css: hobbies chips
- contact.css: contact section and social links
- footer.css: footer styles
- responsive.css: media queries (must be loaded last)

Load order in HTML should match the list above with `responsive.css` last.
